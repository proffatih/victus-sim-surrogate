# Reproducibility archive

Open code, data, and figures associated with a manuscript currently under journal peer review.
The manuscript itself is **not** distributed via this repository; it will be linked from the journal DOI upon publication.

## Layout
- `code/`     — solver and analysis scripts
- `data/`     — inputs / intermediate data
- `results/`  — numerical outputs (CSV / JSON)
- `figures/`  — generated figures (PDF / PNG)
